<!-- ABOUT THE PROJECT -->

Worldwise is an interactive travel tracking application where users can mark cities they‚Äôve visited on a map, view location details, and manage their personal travel history. The project focuses on building map based interactions, managing global state, handling protected routes, and applying performance optimization.

Features:

- Interactive map
- Create, add and delete visited locations
- View city visited city details
- Track user position and center the map dynamically
- Protected routes with user authentication
- Global state management using Context API and useReducer
- Improved bundle size and runtime performance with route based code splitting and lazy loading
